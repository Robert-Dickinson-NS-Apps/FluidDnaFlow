## **The Hard Questions**

### 1. **Is This Actually a CFD Simulation?** *(Critical Issue)*

**The Claim:**
> "Lattice-Boltzmann: Particle-based method simulating fluid as distribution functions."

**The Problem:**

A real Lattice-Boltzmann simulation requires:
- Discrete velocity distributions at each grid point
- Collision and streaming steps per timestep
- Proper boundary conditions (bounce-back for solids)
- Conservation of mass and momentum
- Convergence validation

**What I observe:**
- Parameters change, visuals update
- But does Reynolds number *actually* affect the physics?
- Does viscosity *actually* change flow behavior?
- Or are these just adjusting a pre-rendered animation?

**Tough Question:** If I set Re=10 vs Re=1000, am I seeing *computed* differences in flow regime, or *cosmetic* differences in display?

**What's Missing:**
- No methodology documentation
- No validation against known solutions (flow past cylinder has well-documented behavior)
- No error metrics or convergence indicators
- No source code transparency

**Grade Impact:** Without proof the simulation is real, it's a visualization claiming to be a simulation. That's a credibility problem.

---

### 2. **Scientific Accuracy Concerns** *(High Priority)*

**Issue A: DNA as Flow Obstacle**

Real DNA in fluid flow is:
- **Flexible** (it bends and coils)
- **Microscopic** (nanometer scale)
- **Brownian** (thermal fluctuations dominate)
- **Electrostatically charged** (affects flow interactions)

Your simulation shows:
- Rigid obstacle
- Macro-scale fluid dynamics
- No thermal effects
- No electrostatic considerations

**The Problem:** At actual DNA scales, the Reynolds number would be ~10⁻⁴ to 10⁻⁶ (Stokes flow regime), not 10-1000. The physics shown doesn't match reality.

**Issue B: Parameter Ranges**

| | Parameter | Your Range | Reality for DNA | Mismatch | |
|-----------|------------|-----------------|----------|
| | Reynolds Number | 10-1000 | 0.0001-0.001 | ~10⁶ off | |
| | Velocity | 0.1-5 m/s | μm/s to mm/s | ~10³ off | |
| | Scale | Implied macro | Nanometers | Undefined | |

**Issue C: "DNA Orientation" Control**

Real DNA in solution doesn't maintain fixed orientation—it's constantly tumbling and flexing due to Brownian motion. A static orientation is physically meaningless at molecular scales.
